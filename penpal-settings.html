PenPal AI Writing Assistant

Highlight text anywhere in your browser, click Rewrite with PenPal, and get a polished version back — in your tone, style, and language.

Initial Setup — Complete These Steps First

Choose an AI Provider & Get an API Key

Go to the API & Model tab and select a provider. We recommend OpenRouter — one key gives you access to OpenAI, Anthropic Claude, Google Gemini, and many more. Sign up free at openrouter.ai and create an API key. New here? See Recommended Models below for our suggested free setup.

Paste Your API Key & Test It

On the API & Model tab, paste your key into the API Key field, then click Test Connection to verify it works before saving.

Pick a Model

After entering your key, select a model from the dropdown. Not sure which one? Start with the free recommendation below, then upgrade to a paid model any time for higher-quality results.

Set Your Default Tone & Language

Visit the Writing Style tab to set your default tone from 16 options — from Casual and Witty to Academic and Poetic. Visit Language to set your output language. These become your defaults — you can always override per-use in the PenPal panel.

Save Your Settings

A Save Settings button appears at the bottom of the page whenever you've made a change. Click it to save — a green confirmation banner drops in, and the button disappears again until your next change.

Recommended Models
FREE The easiest way to get started

Provider: OpenRouter  ·  Model: Free Models Router openrouter/free

This is OpenRouter's own free router — instead of picking one specific free model, it automatically selects from whatever free models are available and have capacity at the time of your request. This means it never gets stuck on a rate-limited model. Quality varies depending on which model it picks, but it's a great way to try PenPal AI at zero cost before committing to a paid model.

PAID When you want more consistent quality

Paid models typically cost a small fraction of a cent per rewrite and deliver noticeably more reliable, nuanced results — especially useful for professional or client-facing writing, longer passages, and less common tones or languages.

  • Claude Sonnet 4.6 — Anthropic's well-rounded mid-tier model. A strong everyday upgrade that follows tone and style instructions closely.
  • Claude Opus 4.8 — Anthropic's most capable model. Worth it for nuanced, high-stakes, or longer-form writing where quality matters most.
  • GPT-4.1 — OpenAI's current flagship. A great alternative if you'd prefer to stay within the OpenAI family of models.
  • Gemini 3.5 Flash — Google's fastest frontier model. Excellent multilingual support at a competitive price.

All are available through OpenRouter using the same API key as the free tier — just switch the model on the API & Model tab whenever you'd like better results.

How to Use the Extension

🖱 Right-Click Menu

  1. Select any text on a webpage
  2. Right-click the selection
  3. Click ✏️ Rewrite with PenPal AI in the menu
  4. Review the AI suggestion
  5. Copy or replace text in-place

✏️ Hover Toolbar

  1. Highlight any text on a webpage
  2. A ✏️ Rewrite with PenPal AI button floats above your selection
  3. Click it — AI rewrites instantly using your default tone
  4. Switch tone to auto-regenerate
  5. Copy and Replace sit side by side once the result is ready

⌨ Keyboard Shortcut

  1. Click inside any text field or textarea
  2. Select the text you want to improve
  3. Press Ctrl+Shift+A
  4. The PenPal AI panel opens — use Replace to insert the result directly

🧩 Extension Popup

  1. Click the PenPal AI icon (pencil) in your Chrome toolbar
  2. Paste or type the text you want to improve — or it'll auto-load your page selection
  3. Choose a tone from the row of chips
  4. Review the result, then Copy it back to where you need it

🗗 Pop-Out Window

  1. Open the PenPal AI popup as usual (it loads any highlighted text automatically)
  2. Click the pop-out icon (⧉) next to ⚙ Settings to detach PenPal into its own window
  3. The pop-out stays open while you browse — keep it visible alongside the page you're editing
  4. Highlight new text on any tab and the pop-out updates automatically, as long as you haven't started typing your own text in it
  5. Copy results back into whatever you're working on
PenPal AI works with most text-based websites — social media, email, forums, CRMs, and anywhere you can type in a browser. The Replace button always copies the rewritten text to your clipboard first, then tries to insert it directly into the field (or page text) you highlighted.

If a site's security settings block automatic text insertion, Replace will show "✓ Copied — paste with Ctrl+V" — the text is already on your clipboard, so just click into the field and press Ctrl+V (or ⌘V on Mac).

Google Docs and Microsoft Office 365 use custom editors that block browser extensions from inserting text directly, so Replace will always fall back to copy-and-paste there.

Manage Default Providers
Why this exists: AI providers release new models constantly, and the curated list on the Defaults tab can't keep up the moment something new launches. If a built-in provider has a model that isn't listed yet, add its model ID here and it'll immediately show up in the Model dropdown on the Defaults tab.

Model IDs must match the provider's API exactly (for example OpenRouter uses IDs like anthropic/claude-opus-5, while OpenAI uses IDs like gpt-5). Check the provider's docs or model catalog page for the exact string — even small typos will cause API errors. Use Test Connection on the Defaults tab to confirm it works.

You can also hide built-in defaults you don't use, and override a provider's API endpoint URL if it ever changes. This tab covers the built-in providers only — to add an entirely new provider (e.g. a local server), use 🖥️ Custom Providers.

Privacy & Data Handling
In short: your API key and settings are stored only on your device. Text you choose to rewrite is sent directly to the AI provider you've selected — PenPal AI Writing Assistant does not collect, store, or share it.

API Keys & Local Storage

Your API key (or keys, if you've configured more than one provider) is stored locally in your browser and is used solely to authenticate requests to the provider you've selected. Your keys are never sent anywhere else — not to PenPal AI's developers, and not to any server other than the AI provider's own API.

How Your Content Is Used

When you submit text for rewriting — along with your selected tone, style, and language preferences — that content is sent directly to your chosen provider's API for the sole purpose of generating a rewritten response. PenPal AI Writing Assistant does not independently collect, log, sell, or share this content with any additional third party beyond the provider you've selected and whatever infrastructure is necessary to deliver that request.

Third-Party AI Providers

Each AI provider supported by this extension — including OpenRouter, OpenAI, Anthropic, Google, and any custom endpoint you configure — maintains its own privacy policy, data retention practices, security controls, and terms of service. Once your content reaches a provider, its handling is governed entirely by that provider's policies, not by this extension. We encourage you to review a provider's privacy and data-handling practices before connecting it.

Your Responsibilities

By configuring and using a third-party provider, you acknowledge that content you submit may be processed, logged, or retained by that provider in accordance with its own terms and privacy policy. You are responsible for independently determining whether a provider's practices meet your (or your organization's) legal, regulatory, security, and privacy requirements — particularly when working with sensitive, confidential, or regulated text.
